Detailed Description

This macro will create an OpenCL event called "profilingEvent" that should be used when executing the kernel. If IVW_PROFILING is defined it will call LogInfo with the execution time. Example: IVW_OPENCL_PROFILING(profilingEvent, "") OpenCL::getPtr()->getQueue().enqueueNDRangeKernel(*ernel_, cl::NullRange, globalWorkGroupSize, localWorkGroupSize, nullptr, profilingEvent); Singleton class that manages OpenCL context and queues.
