Pointer to a program no matter if it was succesfully built or not. Creates and builds an OpenCL program. Will automatically reload the program and kernels when the file changes. Only provide the file name as the file will be searched for in all the include paths. The KernelManager will make sure that the processor is not evaluated if building fails.
Note
KernelManager manages pointer memory, do not delete it.
Parameters
fileNameName of file. Added search paths will be used to find the file
