void inviwo::ShaderObject::addOutDeclaration ( std::string  name,
int  location = -1 
)

adds an additional output specifier to the fragment shader The given name will be added as

out vec4 name;

If location index is positive, the output will be 

layout(location = __location__) out vec4 __name__;

Location indices can be reused several times unless more than one output specifier is used.

Parameters
nameidentifier of the output specifier
locationindex location of the output (< MAX_RENDER_TARGETS)
