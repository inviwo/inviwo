Detailed Description

Provides easing functions.

This class provides a function to ease animation between two keys. Different approaches are supported, including overshooting animations.

Member Function Documentation

static double inviwo::animation::Easing::Ease ( const double  t,
EEasingType  HowToEase 
)
inlinestatic

A time input t in [0,1] is assumed. Most easing functions return a value in [0,1] as well. Some functions overshoot, which means you get also values outside of [0,1]. Most interpolations should be fine with that. Linear Interpolation is fine with that.
