Detailed Description

Convenience class for processors that use OpenCL features. Inherit from this class to enable invalidation of processor when a kernel has been compiled.

void inviwo::ProcessorKernelOwner::notifyObserversKernelCompiled ( const cl::Kernel *  kernel)
virtual

Notifies kernel observers that the kernel was compiled and calls invalidate(InvalidationLevel::InvalidResources) on Processor specified in the constructor. Override this method to perform operations after a successful kernel compilation.
