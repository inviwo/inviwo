Detailed Description

template<typename T, typename FactoryObject, typename Key = std::string_view, typename... Args>
class inviwo::StandardFactory< T, FactoryObject, Key, Args >

Standard setup of Factory using factory objects to create an instance. The factory will use a FactoryRegister of FactoryObject and factoryKeyType<Key>. This indirection allows std::string_view to be used as key where as the FactoryRegister will store strings for the keys.

Template Parameters
TThe object created
FactoryObjectAn object with a function std::unique_ptr<T> create(Args...) that can create objects of type T with constructor T(Args...). FactoryObject would usually be a "factory object". A `getClassIdentifier()´ function is also required for registration.
KeyModels a key used to look up T
ArgsA variadic list of arguments passed to T on construction
