KeepOldResults 

Also call done for old jobs, by default old jobs will be discarded.

QueuedDispatch 

Don't submit new jobs while old ones are running. The last submission will be queued and submitted when the current one is finished

DelayDispatch 

Wait for a small delay (500ms) of inactivity before submitting a job.

DelayInvalidation 

Delay invalidation of outports until the job is finished. This will override the default processor invalidation.
