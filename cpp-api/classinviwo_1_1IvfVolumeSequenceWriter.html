Detailed Description

Writer for *.ivfs volume sequence files.

Supports writing a volume sequence to disk. Creates one main file ([name].ivfs) and a series of raw files ([name]xx.raw or [name]xx.raw.gz if zlib compression is available), one for each volume.

The output structure of the ivfs sequence files is: 

<?xml version="1.0" ?>
<InviwoVolume version="2">
    <RawFiles>
        <RawFile content="../data/CLOUDf01.bin.gz">
            <MetaDataMap>
                <MetaDataItem type="org.inviwo.DoubleMetaData" key="timestamp">
                    <MetaData content="1" />
                </MetaDataItem>
            </MetaDataMap>
        </RawFile>
        <RawFile content="../data/CLOUDf02.bin.gz">
            <MetaDataMap>
                <MetaDataItem type="org.inviwo.DoubleMetaData" key="timestamp">
                    <MetaData content="2" />
                </MetaDataItem>
            </MetaDataMap>
        </RawFile>
        ...
    </RawFiles>
    <ByteOrder content="0" />
    <Compression content="1" />
    <Format content="FLOAT32" />
    ...
</InviwoVolume>
